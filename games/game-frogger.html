Frogger

The game Frogger mimics the popular arcade game for your personal computer. The goal of Frogger is to “help a lady frog home” by guiding the frog across a treacherous road and river, and to safety at the top of the screen.
“Despite the lackluster graphics, this game is fun to play. But for Apple owners it constitutes a slap in the face. Sierra On-Line's Atari version of the same game is great. The Graphics sparkle, the sound is wonderful, and the game is certainly In the forefront of that particular market. Why did they release such a mediocre implementation for the market that put On-Line where it is now?”

Impact
Frogger was created for the Apple II by Olaf Lubeck in 1982 and released in 1983. It was published by Sierra On-line.

Help

The Game page contains game play information, a link to a manual (if available), and the impact and history of the game. This is where you will upload and play the game you downloaded on the Magazine page.


Playing a Game
  1. Select Load Disk on the Apple II emulator (Your File/Finder window will appear)
  2. Select the game .dsk file in the Download folder in your File/Finder window
  3. Click Open in the File/Finder window after selecting the game .dsk file
  4. Play the game in the Apple II emulator
